Web browsers receive HTML documentsfrom a web server or from loaval storage and render them into multimedia web pages. HTML describes the strucure of a web pages semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
